# Example 03: Fibonacci — All Approaches

## Problem
Compute the nth Fibonacci number using 4 different methods and compare.

## Methods
1. Naive recursion — O(2^n) time
2. Memoization — O(n) time, O(n) space
3. Bottom-up DP — O(n) time, O(n) space
4. Space-optimized — O(n) time, O(1) space

## Input: n = 40
## Output: 102334155
