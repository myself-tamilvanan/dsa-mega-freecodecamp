# Example 02: Space Complexity Comparison

## Problem
Compare O(1) vs O(n) vs O(n²) space usage and understand when to optimize.

## Input
- n = 1000

## What to Observe
- In-place sort uses O(1) extra space
- Memoization uses O(n) space
- 2D DP table uses O(n²) space

## Key Insight
Space-time tradeoff: using more space often speeds up time.
