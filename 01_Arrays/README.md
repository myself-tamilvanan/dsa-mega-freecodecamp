# Chapter 01: Arrays
Arrays store elements in contiguous memory with O(1) index access.

## Key Patterns
- **Hash Map**: O(1) lookup — Two Sum, Contains Duplicate, Anagram
- **Frequency Count**: Counter() — Top K, Valid Anagram
- **Prefix/Suffix**: product except self, range sum
- **Sorting first**: simplify pair/group problems
- **Cycle Sort**: find missing/duplicate in [1..n]

## Complexity
| Op | Avg | Worst |
|----|-----|-------|
| Access | O(1) | O(1) |
| Search | O(n) | O(n) |
| Insert end | O(1) | O(n) |
| Insert mid | O(n) | O(n) |
| Delete mid | O(n) | O(n) |
