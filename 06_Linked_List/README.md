# Chapter 06: Linked List
Nodes linked by pointers. O(1) head insert, O(n) access.

## Key Techniques
- **Dummy node**: simplifies edge cases
- **Fast & Slow pointers**: cycle, middle, nth-from-end
- **In-place reversal**: 3-pointer (prev, curr, next)
- **Merge**: combine two sorted lists
