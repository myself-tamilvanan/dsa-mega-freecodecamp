# Chapter 11: Graphs
Vertices + edges. Represent as adjacency list (dict of lists).

## Algorithms
| Algorithm | Use | Time |
|-----------|-----|------|
| BFS | Shortest path (unweighted) | O(V+E) |
| DFS | Connected components, cycle | O(V+E) |
| Union-Find | Components, MST | O(α(n)) |
| Dijkstra | Shortest path (weighted) | O(E log V) |
| Topological Sort | DAG ordering | O(V+E) |
| Kruskal | MST | O(E log E) |
