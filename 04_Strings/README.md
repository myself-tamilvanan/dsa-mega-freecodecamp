# Chapter 04: Strings
Strings are immutable sequences. Use join() instead of + for efficiency.

## Key Techniques
- **Two pointers**: palindrome, reversal
- **Hash/sort for anagrams**: canonical form
- **Expand around center**: palindromic substrings
- **Sliding window**: substring problems
- **Split/join**: word manipulation
