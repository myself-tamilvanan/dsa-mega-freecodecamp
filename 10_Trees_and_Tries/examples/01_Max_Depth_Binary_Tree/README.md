# Example 01: Maximum Depth of Binary Tree (LC 104)

## Problem
Find the maximum depth (number of nodes along the longest root-to-leaf path).

## Input
Tree: [3,9,20,null,null,15,7]
## Output
3

## Approaches
1. Recursive DFS — O(n) time, O(h) space
2. Iterative BFS (level-order) — O(n) time, O(n) space
