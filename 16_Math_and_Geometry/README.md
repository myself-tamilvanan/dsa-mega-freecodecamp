# Chapter 16: Math & Geometry

## Key Topics
- **Modular arithmetic**: (a+b)%m = ((a%m)+(b%m))%m
- **GCD/LCM**: Euclidean algorithm O(log min(a,b))
- **Sieve of Eratosthenes**: all primes ≤ n in O(n log log n)
- **Fast Exponentiation**: x^n in O(log n)
- **2D rotations**: 90° CW = (row,col)→(col,n-1-row)
