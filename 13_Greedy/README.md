# Chapter 13: Greedy
Make locally optimal choice at each step.

## When Greedy Works
- Greedy Choice Property: local optimal → global optimal
- Optimal Substructure: optimal solution has optimal subproblems

## Common Patterns
- Sort by end time (interval scheduling)
- Sort by ratio/value (fractional knapsack)
- Always pick largest/smallest available (heap)
